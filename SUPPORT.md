# Support HEStats

HEStats is intended to be open public infrastructure for UK higher education financial analysis.

## Ways To Help

- Submit data corrections with official sources.
- Add missing annual report links.
- Improve documentation and methodology notes.
- Fix bugs or add tests.
- Review pull requests.
- Sponsor the project on GitHub.
