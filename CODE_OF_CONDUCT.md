# Code of Conduct

HEStats is a public-interest open-source project. Contributors should be able to participate without harassment, intimidation, or bad-faith conduct.

## Expected Behavior

- Be respectful and constructive.
- Assume good intent, but prioritize evidence and clarity.
- Give feedback on work, not identity.
- Be precise when discussing data quality, uncertainty, and sources.
- Welcome new contributors and help them make useful contributions.

## Unacceptable Behavior

- Harassment, threats, or discriminatory language.
- Personal attacks or repeated hostile comments.
- Publishing private information without permission.
- Deliberately submitting false, unsourced, or misleading data.
- Derailing issue threads after maintainers ask to return to scope.

## Enforcement

Maintainers may edit, hide, or remove comments; close issues; reject pull requests; or block participants who violate this code of conduct.

If you need to report a conduct issue, email the maintainer listed in the repository profile or use a private GitHub contact route if available. Include links, screenshots, or context where possible.
